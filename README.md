## 🧭 景点详情页功能

此页面播放背景音乐（BGM）节省空间，也可以拓展为播放介绍词或导游词等音频。  
点击收藏按钮可将景点加入收藏页面，后续可以扩展为类似“购物车”的功能（统一购票）。
